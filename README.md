## About Me

I'm a CS senior at Morgan State University (May 2026) and an incoming M.S. AI student at Texas A&M. I build full-stack AI systems, self-host my own infrastructure, and ship real products, not just side projects.

My homelab runs 24/7 on a Ryzen 7 5800X with 32GB RAM, hosting LLM inference, vector search, RAG pipelines, and everything I'm currently building. I use it as my personal dev platform and deploy things people actually use.

- Currently building: **AI Resume Tailoring System** (self-hosted RAG + Claude Haiku) and **BearAdvisorAI** (Gemini 1.5 Pro + Vertex AI capstone)
- Researching: LLM failure modes, prompt evaluation, and automated grading systems
- Open to: Software Engineer / AI Engineer roles in Healthcare Tech and Fintech
- Writing about building on the homelab at: camwillm.hashnode.dev
